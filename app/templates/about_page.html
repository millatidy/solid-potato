Hello!

This web application was developed by Milla Mutiwokuziva as part of a Britecore's Remote Engineering job application process.

The Tech

As per requirements, this project's backend is developed in python using Flask as the web Framework of choice. The frontend is developed with HTML, CSS(Bootstrap 4), and JS(JQuery for RESTAPI calls, and a bit of rendering and Knockout.js for data handling). This web app is deployed on an AWS EC2 t2.mirco instance, running Unbuntu 18.04, therefore Elasticsearch is disabled due to memory limitations. The web app sits in a docker container behind a Gunicorn WSGI Server which sits behind an Nginx reverse proxy server. Database of choice is PostgreSQL also running on AWS. Errors are logged by a Raven from Sentry and are logged on the sentry website.

The Features

Besides what is required by Britecore team the only extra additional feature at this point is the ability to search Features in the application.
